# esp32 学习与笔记与草稿项目

个人 esp32 学习与笔记与草稿项目，不是软件项目。

esp32、arduino、PlatformIO、vscode、opencode

## 目录约定

```
note-EAPVO/              # 学习笔记（markdown 格式）
- index.md          #   笔记索引，新笔记完成后更新此文件
- 草稿260622/ # 偏草稿的记录，其中是一些随意命名的目录
  - 260611-xxxxxx/ # 其中可以是目录也可以是md文件（数字即日期编号 年月日-文字）
    - 0611-1535-xxxxx/ # 其中可以是目录也可以是md文件（数字即日期编号 月日-时分-文字）
      - 0622-1437-xxxxxxxxxx.md
      - ……
    - 0622-1536-xxxxxxx.md
    - ……
  - 260622-xxxxxxxx/
  - ……
- 笔记260622/ # 偏正式的笔记，其中是一些随意命名的md文件
  - 260611-xxxx.md
  - 260622-xxxxxx.md
  - ……
```

## 提交信息规范

前缀使用标准英文，内容使用中文，如：
- `docs(tool-learning): 补充 xxxx 安装和配置说明`
- `feat(experiments): 添加 xxx 实验代码`
- `fix(notes): 修正 xxx 链接错误`
